# RevGen
**RevGen** _is a Reverse Shell Generator and Listener_

## Installation
```
git clone https://github.com/shexer/RevGen
You must install netcat.
```
## Useage
```
python3 RevGen.py
```
## Binaries For Reverse Shell
```
1)Bash            6)Python
2)Netcat          7)Ruby
3)Perl            8)Socat
4)Pip             9)Telnet
5)Php             10)Vim
```
